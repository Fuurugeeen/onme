# OnMe - UI Design Brief

AIデザインサービス向けの画面イメージ生成依頼書

---

## 1. プロダクト概要

### サービス名
**OnMe** (オンミー)

### コンセプト
SNSやショート動画に時間を奪われがちな学生のための、自己理解AIコーチングサービス

### キャッチコピー
「自分を知る、毎日の対話」

### 核となる価値
- **比較しない場所** - 他人との比較は一切しない。昨日の自分との比較のみ
- **安心できる避難所** - SNSの比較文化から離れる安全な場所
- **AIが寄り添う** - フラットで親しみやすいAIコーチ

---

## 2. ターゲットユーザー

### メインターゲット
- **年齢**: 高校生〜大学生（16-22歳）
- **特徴**:
  - SNS・ショート動画で気づけば時間を溶かしてしまう
  - 他人と比較して落ち込みやすい
  - 自分の強みや特性が分からない
  - 将来に漠然とした不安がある

### ユーザーのペルソナ
> 大学2年生のユウキ。毎日TikTokを見て3時間が過ぎている。友達のインスタを見て「みんな充実してるな...」と落ち込む。就活を意識し始めたけど、自分の強みが分からない。自己分析の本を買ったけど続かなかった。

---

## 3. デザインの方向性

### トーン & ムード
| 要素 | 方向性 |
|------|--------|
| 雰囲気 | 穏やか、安心感、プレッシャーのない |
| テンション | 落ち着いた、でも温かみがある |
| 印象 | シンプル、クリーン、ミニマル |
| 避けたいもの | ギラギラ、競争感、煽り、過度な装飾 |

### カラーパレット（現在の設定）
| 用途 | HSL値 | 説明 |
|------|-------|------|
| Background | 0 0% 100% | 純白 |
| Foreground | 222.2 84% 4.9% | ほぼ黒（ダークネイビー） |
| Primary | 222.2 47.4% 11.2% | 落ち着いたネイビー |
| Secondary | 210 40% 96.1% | 薄いグレーブルー |
| Muted | 210 40% 96.1% | 控えめなグレー |
| Accent | 210 40% 96.1% | アクセント |
| Destructive | 0 84.2% 60.2% | 警告用レッド |

### カラー変更の希望
- 現在のshadcn/uiデフォルトパレットからの変更可
- **希望の方向性**:
  - 温かみのある優しい色合い（例：ペールブルー、ラベンダー、ミントグリーン）
  - または、シンプルな白ベースにアクセントカラー1色
  - 学生向けなので、堅すぎない・ビジネスライクすぎない

### 参考にしたいテイスト
- Headspace（瞑想アプリ）- 穏やかで温かみのあるイラスト
- Notion - シンプルでクリーンなUI
- Duolingo - 親しみやすいがうるさくない
- LINE - チャットUIの使いやすさ

## 4. 画面一覧と詳細

### 4.1 Login画面
**パス**: `/login`

**目的**: Google認証でログイン

**レイアウト**:
```
┌─────────────────────────────┐
│                             │
│         OnMe ロゴ            │
│                             │
│   "自分を知る、毎日の対話"    │
│                             │
│   ┌───────────────────┐     │
│   │ 🔵 Googleでログイン │     │
│   └───────────────────┘     │
│                             │
└─────────────────────────────┘
```

**要素**:
- サービスロゴ（シンプルで覚えやすい）
- キャッチコピー
- Googleログインボタン

**雰囲気**:
- 清潔感があり、安心感を与える
- 余白を大きくとり、圧迫感のない

---

### 4.2 Onboarding画面
**パス**: `/onboarding`

**目的**: AIとの初期対話で自己理解のきっかけを作る

**レイアウト**:
```
┌─────────────────────────────┐
│  ← 戻る                      │
├─────────────────────────────┤
│                             │
│   💬 AI との対話エリア        │
│   （自己紹介・価値観の発見）   │
│                             │
├─────────────────────────────┤
│  [入力欄]            [送信]  │
└─────────────────────────────┘
```

**要素**:
- チャット形式のUI
- AIメッセージ（左寄せ、吹き出し）
- ユーザーメッセージ（右寄せ、吹き出し）
- テキスト入力欄と送信ボタン

**AIのトーン**:
- 「はじめまして！OnMeへようこそ」
- 「君のことを少し教えてくれる？」
- フラットで友達のような口調

│                             │
│  🔥 継続 12日目              │
│                             │

│  │   しましょう」         │  │
│  │                       │  │
│  │      [話す →]         │  │
│  └───────────────────────┘  │
│                             │
│  📝 最近の気づき             │
│  ┌───────────────────────┐  │
│  │ • 朝の時間が集中できる  │  │

**ポイント**:
- 開いた瞬間に「今日やること」が分かる
- プログレスバーは達成感を演出
- 「話す」ボタンは最も目立つ位置に

## 5. 技術的な制約

| 項目 | 内容 |
|------|------|
| プラットフォーム | Webアプリ（モバイルファースト） |
| 画面サイズ | スマートフォン中心（375px〜428px幅） |
| CSSフレームワーク | Tailwind CSS |
| UIライブラリ | shadcn/ui パターン |
| アイコン | 絵文字またはシンプルなラインアイコン |

## 6. デザイン生成の依頼

### 希望する成果物
1. **各画面のモックアップ**（6画面）
2. **カラーパレット提案**
3. **ロゴデザイン案**（シンプル、覚えやすい）

### 生成時のキーワード
```
mobile app UI, clean minimal design, soft colors,
friendly coaching app, student-focused,
calming atmosphere, Japanese text,
chat interface, dashboard layout,
no comparison culture, safe space feeling
```

### 避けてほしい要素
- 競争を煽るデザイン（ランキング、他者比較）
- 派手すぎる色使い
- 複雑なアニメーション
- ビジネスライクで堅い印象
- SNS的なフィード形式

## 8. 補足情報

### サービス名「OnMe」の由来
「On Me」= 自分自身に向き合う、自分について

### 想定利用シーン
- 朝起きてすぐ（5分の振り返り）
- 寝る前（今日の自分を振り返る）
- SNSを見そうになった時の代替

### 大切にしたいUX
> アプリを開くと、温かく迎えてくれる友達がいる感覚。
> 「今日も来てくれたんだね」と言ってくれる安心感。
> 何も達成できなくても責められない場所。
